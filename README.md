# 🐓 Heckin' Peckin'

## Controls
To walk to right = D/➡️

To walk to left = A/⬅️

To jump = Space bar (hold to jump higher)

To double jump = Double tap space bar

To wall jump = Jump onto wall (space bar), then use right/left between walls
